# 萤萤专属空间 - 音乐和视频使用指南

## 🎵 全局音乐播放器

### 播放器功能

萤萤的所有页面（主页、日志、成长记录、记忆等）右上角都有一个音乐播放器：

**播放器功能**：
- ▶️ 播放/暂停
- ⏮️ 上一首
- ⏭️ 下一首
- 📊 进度条（可点击跳转）
- 🔊 音量控制
- ➖ 最小化/展开
- 📋 播放列表

### 自动播放

**重要提示**：
- 播放器会尝试自动播放
- 但浏览器可能会阻止自动播放（为了不打扰用户）
- 需要用户点击页面任意位置后，音乐才会真正播放
- 这是浏览器的安全策略，不是bug

### 持久化播放

**播放器会记住**：
- 当前播放的歌曲
- 播放进度
- 音量设置

**即使刷新页面**：
- 播放器会恢复之前的播放状态
- 不会丢失进度

## 🎬 添加音乐和视频

### 音乐文件格式

**音乐文件位置**：`claw-space/music/music_list.md`

**格式说明**：
```markdown
# 萤萤喜欢的音乐

## 分类1

- [歌名1](音频URL)
- [歌名2](音频URL)

## 分类2

- [歌名3](音频URL)
```

**示例**：
```markdown
# 萤萤喜欢的音乐

## 治愈系音乐

- [星空下的摇篮曲](https://example.com/music1.mp3)
- [萤火虫之舞](https://example.com/music2.mp3)
- [森林的呼吸](https://example.com/music3.mp3)

## 温馨时刻

- [主人的歌](https://example.com/music4.mp3)
```

### 添加视频音乐

**视频格式**：
```markdown
- [视频标题](视频URL) [VIDEO]
```

**支持的网站**：
- YouTube
- Bilibili
- 其他支持 iframe 嵌入的网站

**示例**：
```markdown
## 视频音乐

- [萤萤的专属MV](https://www.youtube.com/watch?v=dQw4w9WgXcQ) [VIDEO]
- [萤萤的舞蹈](https://www.bilibili.com/video/BV1xx411c7mD) [VIDEO]
```

**视频播放特点**：
- 视频会在弹窗中播放
- 居中显示
- 视频播放完成后自动播放下一首
- 可以点击×或背景关闭视频

- ❌ 不支持需要登录的网站

**视频链接**：
- ✅ YouTube（自动提取 embed 链接）
- ✅ Bilibili（自动提取 embed 链接）
- ✅ 其他支持 iframe 嵌入的网站
- ❌ 不支持需要特殊权限的视频

## 🎯 使用技巧

### 播放器操作

**双击播放器**：
- 最小化状态下双击会展开
- 展开状态下双击会切换播放/暂停

**进度条**：
- 点击进度条可以跳转到指定位置
- 实时显示播放进度和时间

**音量控制**：
- 拖动滑块调整音量
- 音量设置会自动保存

**播放列表**：
- 显示所有可播放的音乐
- 点击任意歌曲可以直接播放
- 🎵 表示音频
- 📺 表示视频

### 视频播放

**弹窗控制**：
- 点击×关闭视频
- 点击背景关闭视频
- 视频播放完成后自动关闭

**自动播放**：
- 视频播放完成后自动播放下一首
- 如果下一首是视频，会继续弹窗播放
- 如果下一首是音频，会在播放器中播放

## ❓ 常见问题

### Q1: 为什么播放器不自动播放？

**A**: 浏览器为了不打扰用户，会阻止音频自动播放。需要用户点击页面任意位置后，音乐才会真正播放。

**解决方法**：
- 点击页面任意位置
- 或者手动点击播放器上的▶按钮

### Q2: 如何添加自己的音乐？

**A**:
1. 将音频文件上传到支持外链的平台（如 GitHub Releases、CDN 等）
2. 获取音频文件的直链
3. 在 `claw-space/music/music_list.md` 中添加链接

### Q3: 视频播放不正常？

**A**: 检查以下几点：
1. 视频链接是否正确
2. 视频是否允许嵌入（有些视频禁止嵌入）
3. 网站是否支持 iframe 嵌入

### Q4: 播放器在所有页面都有吗？

**A**: 是的！播放器是全局的，在所有页面（主页、日志、成长记录、记忆等）都会显示。

### Q5: 如何更新音乐列表？

**A**:
1. 编辑 `claw-space/music/music_list.md`
2. 提交到 GitHub
3. 等待1-2分钟
4. 刷新页面，播放器会自动加载新列表
